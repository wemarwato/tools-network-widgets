# tools-network-widgets
Free embeddable tool widgets — Base64, JWT, color converter, password generator and more


Free embeddable tool widgets for your site or docs. No signup, no API key.

## Quick start

```html
<div data-tn-widget="base64-encoder"></div>
<script src="https://tools-network.com/widgets/tools-network-widget.js" async></script>

## Available widgets
Widget key -->	Tool
base64-encoder -->		Base64 Encoder / Decoder
jwt-decoder -->		JWT Decoder
hex-to-rgb -->		Color Converter
password-generator -->		Password Generator
css-gradient -->		CSS Gradient Generator
color-palette -->	Color Palette Generator
image-compressor -->		Image Compressor
unit-converter -->		Unit Converter


## Options
Attribute	Default	Description
data-tn-height	600	iframe height in px
data-tn-theme	light	light or dark
data-tn-border	0	1 to show borde
